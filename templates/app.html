Event Assignment Optimizer

This tool will take practice test data and find the best possible team that can be formed based on the following criteria:

  • Up to 10 members can be selected total.
  • Each event can have at most 5 members competing in it.
The program may suggest that a member do an event they did not practice for, but it assumes that they would score 0 points on any events they did not practice for.

Requirements:

  • First row has column headers (name, event 1, event 2, ..., event 6)
  • Every following row has the appropriate data for one member
  • No extraneous rows (additional columns may be okay)
  • The program currently only supports up to 3 scores as input per member.
